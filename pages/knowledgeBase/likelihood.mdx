# 似然估计

[b 站入门视频，非常易懂](https://www.bilibili.com/video/BV1CR4y1L7RC/?spm_id_from=333.337.search-card.all.click&vd_source=6b74b140d977ca4e80c1459bc59f5005)

假设我们有一个硬币，我们想知道它正面朝上的概率。我们进行了 10 次抛硬币实验，结果是 7 次正面朝上。

<span style={{ color: "orange" }}>频率</span> 是 7/10，即 70%。\
<span style={{ color: "orange" }}>似然</span> 则是指在观察到 7 次正面朝上的情况下，硬币正面朝上的概率为
p 的可能性。

于是乎，似然函数如下：

```
L(p) = p^7 * (1-p)^3
```

当 p = 0.7 时，L(p) = 0.7^7 _ 0.3^3 ≈ 0.023 \
当 p = 0.5 时，L(p) = 0.5^7 _ 0.5^3 ≈ 0.0078

### 最大似然估计

有了这个似然函数，我们想要找到让似然最大的 $ p_0 $，这样意味着在 $ p_0 $ 概率下，7 次正面朝上的可能性最高。
于是对 L(p) 求导，找到最大值点 $ p_0 = 0.7 $，即最大似然。
